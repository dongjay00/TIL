# 문항번호 2

> - 소요시간: 3분



## 문제

### 문제 설명

ul, li, ol 요소에 대해 알고 있는지 물어보는 문제입니다.



### 제한 사항

HTML 마크업에서 “ • “ , 숫자 기호 텍스트를 포함하지 말고 개발해주세요



### 가이드라인

ul, ol 의 차이를 알아봅시다.





## 나의 답안

```html
<ul>
    <li>사과</li>
    <li>오렌지</li>
    <li>포도</li>
    <li>체리</li>
</ul>

<ol>
    <li>가지</li>
    <li>감자</li>
    <li>당근</li>
    <li>대파</li>
</ol>
```
